## Customization Tips

### Changing Colors
Edit `/app/globals.css`:
```css
--primary: oklch(0.4 0.15 280);    /* Your primary color */
--accent: oklch(0.65 0.15 280);    /* Your accent color */
```

### Adjusting Animations
In component files:
```tsx
duration: 0.8,          // Shorten/lengthen
ease: 'power3.out',     // Change easing
delay: 0.1,             // Adjust stagger
```

### Updating Content
In each component:
```tsx
<h1>Your custom heading</h1>
<p>Your custom description</p>
```
